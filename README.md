# PantryLog (대용량 식료품 재고 & 소분 관리 시스템)

> **"소분할 때 깨닫는 본품 품절 비상사태는 이제 그만!"**  
> PantryLog는 가정에서 작은 병에 소분하여 사용하는 대용량 조미료 및 식료품의 **본품 남아있는 용량을 직관적으로 추적**하고, **원클릭 소분 차감 및 부족 재고 자동 알림/장보기 목록 관리**를 제공하는 실생활 밀착형 웹 서비스입니다.

🌐 **Vercel 배포 주소**: [https://pantrylog-xi.vercel.app](https://pantrylog-xi.vercel.app)

---

## 🌟 개발 배경 및 해결하는 문제

대부분의 가정에서는 대용량 조미료(간장 1.8L, 참기름, 식초 등)를 그대로 사용하지 않고 **작은 소분 용기에 담아 사용**합니다.  
평소에는 소분용 병만 보다가, 소분용 병이 비어서 원래 대용량 용기에서 채우려 할 때:
> *"어? 원래 대용량 병도 다 썼네?!"*

하는 긴급 상황이 발생하여 요리 계획이 틀어지거나 급하게 마트를 다녀와야 합니다.

**PantryLog**는 유통기한 중심 관리가 아닌, **현재 집에 남아 있는 본품 식료품의 용량을 직관적으로 추적**하여 미리 재구매 시점을 놓치지 않도록 돕습니다.

## 🔑 핵심 기능

1. **대용량 식료품 재고 시각화**
   - 진행률 바(Progress Bar), 퍼센트(%), 숫자 용량(`920 / 1800 mL`), 🟢 충분/🟡 부족/🔴 구매 필요 뱃지 한눈에 확인

2. **💡 원클릭 '소분하기' (Decant) 기능**
   - 소분용 병을 채울 때 카드에서 **`[소분하기]`** 버튼 클릭 한 번으로 미리 지정한 1회 소분량(예: 100mL, 200mL)만큼 본품 재고 자동 차감!

3. **직관적인 현재 용량 직접 수정**
   - 소분 이외에도 보관함을 확인하고 슬라이더 또는 숫자로 쓱 수정하여 실제 잔여량 업데이트

4. **🛒 자동 장보기 목록 (Shopping List)**
   - 최소 재고 기준 이하로 떨어진 품목이 자동으로 장보기 목록에 집계
   - 장보기 완료 후 원클릭 재충전(Refill) 지원

5. **카테고리 & 검색 & 필터링**
   - 조미료/소스, 기름/장류, 면/통조림, 냉동식품 등 분류별 조회
   - 구매 필요 품목만 모아보기, 잔여 비율순 정렬
